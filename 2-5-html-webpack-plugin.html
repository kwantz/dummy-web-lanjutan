        Jika ingin menambah file .html yang baru, cukup menambah new HtmlWebpackPlugin()

        bagaimana cara memilih file .js yang diinginkan dalam 1 file .html
      

      


      
Jalankan perintah npm run build dan hasilnya adalah seperti berikut:

      

      

      

      


      
Beberapa option yang tersedia pada HtmlWebpackPlugin

      

        

          title
          title yang digunakan untuk .html yang dibuat.
        

        

          filename
          nama file .html yang diinginkan. Secara default bernama index.html. Juga dapat digunakan untuk subdirectori (misalnya: asset/admin.html)
