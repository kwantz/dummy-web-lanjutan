Arrow Function

arrow function (=>) membantu dalam mendeklarasikan async function secara lebih singkat lagi dan memang ada saatnya dimana kita benar-benar harus memakai arrow function. Dengan menggunakan arrow function, penulisan syntax seperti forEach ataupun fungsi map() akan menjadi semakin mudah dan simpel.

Enchanced Object Literals

      

        Object Literal memudahkan dalam membuat object dengan cepat dan mudah.
        Untuk membuat object, hanya cukup menggunakan daftar kunci: value pairs
        yang dipisahkan oleh koma. Pada ES6, terdapat 3 hal dalam deklarasi objek literal:
      

      
    
        
  1. sintaks singkatan untuk menginisialisasi properti dari variable.
    2. 
        
  2. sintaks singkatan untuk mendefinisikan metode fungsi.
    3. 
        
  3. kemampuan untuk menghitung nama properti dalam definisi literal object.
    4. 
      

      

      

      

        pada contoh diatas, kita membuat sebuah function getLaptop, disini kita
        membuat function untuk memudahkan pass variabelnya. Jadi dapat kita lihat apabila
        menggunakan ES5, maka object yang dikembalikan dibuat menggunakan literal object.
        Properti object ini dibuat dengan menetapkan nilai parameter yang dilewatkan ke
        coressponding keynya. Dan pada ES5, dapat kita lihat bahwa ada perulangan. Dan pada ES6, dia menghapus semua perulangan itu.
        Apa yang terjadi pada ES6 adalah ia memeriksa apakah property keynya memiliki nama variabel yang sesuai dan kemudian menetapkan
        nilai variabel tersebut kedalam propertynya. Perhatikan bahwa jika tidak ada variable yang memiliki nama yang sama dengan property key
        yang ditentukan, maka akan terjadi error.
      

      

      

      
pada contoh diatas, kita tidak memerlukan function untuk mendefine sebuah method.

      

      

      

        pada contoh diatas, dapat dilihat bahwa value dari variabel 'name' adalah
        'make', namun disini 'make' juga digunakan sebagai nama property,
        sehingga kita dapat mengakses property menggunakan 'laptop.make'. Dan pada contoh
        diatas juga, value dari 'name' dan 'i' digabung untuk menghasilkan
        sebuah nama property.
