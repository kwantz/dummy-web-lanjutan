Konfigurasi Bahasa Lain

TypeScript

Pertama sekali kita harus menginstal dependensinya yaitu npm install --save-dev typescript ts-node @types/node @types/webpack. Setelah itu berikut ialah contoh penulisan file konfigurasi webpack.config.ts: 


      

        Setelah itu, kita perlu membuka file tsconfig.json dan mengganti target pada compilerOptions ke ES5
        dan module yaitu CommonJS (atau hilangkan opsi module). Cara ini lebih singkat dibandingkan cara lain.
